A mockin' Roc REPL!

This is a mock-up of a Web REPL for Roc. But instead of the actual Roc language, it only accepts numbers 0-255! It "compiles" a Wasm module that counts down from that number, and runs it.

The same web page should work with minimal adjustments whenever we manage to get a WebAssembly build of the real Roc compiler working. But it was a lot easier to debug with a tiny mock compiler!

How it works

  • Two Wasm modules: a fake "compiler", and a fake "app"
  • The "compiler" is a little program that contains an array of bytes, and a function that can modify exactly one of those bytes. The other bytes are just constant numbers... that "just happen" to form a valid WebAssembly module, because I cheated.
  • JS calls the "compiler" with your input, and it sets the editable byte of the app to the value you chose. JS then reads the whole byte array, creates a WebAssembly instance from it, and runs it.
  • The app returns an array of bytes representing all the numbers in the countdown.
  • JS transforms those numbers to a string to display
