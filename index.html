The mockin' Roc REPL!

This is a mock-up of a Web REPL for Roc. But instead of the actual Roc language, it only accepts numbers 0-255! It "compiles" a Wasm module that counts down from that number to zero, and runs it.

Still, most of the infrastructure is similar to what we need for the real Web REPL

How it works

  • Two Wasm modules: a fake "compiler", and a fake "app"
  • The "compiler" contains an array of bytes that it can modify, based on your input, and pass to JS
  • JS takes those bytes, turns them into a WebAssembly instance, and runs it
  • The "app" returns an array of bytes representing all the numbers in the countdown from your number to zero. (Your input is a compile-time constant from the app's point of view.)
  • JS transforms those numbers to a string and displays it
